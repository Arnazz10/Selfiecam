## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Arnazz10/Selfiecam.git
   cd Selfiecam
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Navigate to `http://localhost:5173` (or the URL shown in your terminal).

   *Note: You must allow camera and microphone permissions for the app to function correctly.*
